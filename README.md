# AION2-DPS-VIEWER

아이온2용 `DPS 미터 + 아툴 정보 조회 + 전투 기록` 통합 프로그램입니다.

`DPSVIEWER`는 게임 화면 위에 오버레이처럼 띄워 실시간 DPS를 확인하고, 감지된 캐릭터의 정보와 전투 기록을 함께 볼 수 있도록 만든 WPF 데스크톱 앱입니다.

## 현재 버전

- `v4.0.14`
- 기준 날짜: `2026-03-17`

## 주요 기능

- 실시간 DPS 집계
- 보스 대상 추적 및 보스 DPS 필터
- 자동 감지 캐릭터 정보 조회
- 전투 기록 저장 및 조회
- 간소화 오버레이 모드
- 전역 단축키 지원
- 로그 전송 및 DPS 스냅샷 로그 저장

- 공식 사이트: [Npcap](https://npcap.com/)

- 직접 설치: [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0/runtime)
